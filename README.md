# Movie-recommendation-system

OVERVIEW OF THE PROJECT

This is a project based on machine learning in which the model gives the top 5 suggestions of the movies which the user gives in the input.
The idea is simple just to build a recommendation system which will recommend or suggest the things on the basis of our input.

Prerequisites

b) pycharm community edition

  4) pip install scikit-learn
  5) pip install nltk
  6) pip install streamlit
  7) pip install pickle
  8) pip install keras
